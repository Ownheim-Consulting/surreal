# NASA_Space_Apps_2022_Backend
Backend Code for NASA Space Apps 2022.

# Dependencies
- Python 3.10
- Pipenv

# How to Run
This project uses pipenv for dependency management. 

1. Install pipenv
```
pip install --user pipenv
```

2. Begin pipenv shell
```
pipenv shell
```

3. Install project's dependencies
```
pipenv install
```

4. Run the project
```
python src/main.py
```
